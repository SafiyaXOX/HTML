Introduction to HTML

HTML stands for HyperText Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

HTML elements are represented by tags. HTML tags label pieces of content such as "heading", "paragraph", "table", and so on. Browsers do not display the HTML tags, but use them to render the content of the page.

Tables

Tables are used to display data in a structured format. We use the <table> tag to create a table. The <tr> tag is used to create a table row. The <th> tag is used to create a table header. The <td> tag is used to create a table cell.

Lists

HTML supports several types of lists that help organize content. The two primary types are unordered lists (bulleted lists) and ordered lists (numbered lists). Lists are helpful for structuring information clearly.

Example:

Unordered List

  • Item One
  • Item Two
  • Item Three

Ordered List

  1. First Step
  2. Second Step
  3. Third Step

Use the <ul> tag for unordered lists and <ol> for ordered lists. Each list item goes inside a <li> tag.

Nested Lists

You can nest lists inside other lists to create sub-items. Here, we used an unordered list inside another unordered list. You can also mix ordered and unordered lists if needed.

    • URLs in HTML

    URLs are used to link to other web pages. We use the <a> tag to create a link. This tag <a> has the following attributes:

    • href: Specifies the URL of the link
    • target: Specifies where to open the linked document
    • title: Specifies the title of the link
    • rel: Specifies the relationship between the current document and the linked document
    • type: Specifies the MIME type of the linked document. MIME stands for "Multipurpose Internet Mail Extensions".
    • charset: Specifies the character set of the linked document
    • name: Specifies the name of the link
    • rev: Specifies the reverse link of the link

    Forms in HTML

    Forms are used in HTML to collect user input and send it to a server for processing. They enable interactive web pages.

    Common use Cases:

    • User Authentication- Login and registration.
    • Data Collection- Contact forms, surveys, feedback.
    • E-Commerce- Checkpuut, payment, shipping.
    • Search- Search boxes.
    • Settings/Preference- User profiles, account settings.
    • File Uploads- Document and image uploads.
    • Subscriptions- Newsletter signups, email lists.

    How forms work:

    • User Input- Fields like text input, checkboxes, dropdowns, etc.
    • Validation- Check data before submission (client-side).
    • Submission- Send data to a server (via GET or POST).
    • Processing- Server processes the data (save to database, send email, etc.)
    • Response- Show a success message or redirect
