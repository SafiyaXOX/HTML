Introduction to HTML

HTML stands for HyperText Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

HTML elements are represented by tags. HTML tags label pieces of content such as "heading", "paragraph", "table", and so on. Browsers do not display the HTML tags, but use them to render the content of the page.

HTML consists of headings. The headings go from <h1> to <h6>. This text is written in what you call a paragraph. Noted using <p> and </p> tags."
The line breaks you see (ending a line and starting a new one) is noted using <br> tag.

Below are the types of headings in HTML.

This is an h1 heading.

This is an h2 heading.

This is an h3 heading.

This is an h4 heading.

This is an h5 heading.
This is an h6 heading.
So you can see that the headings decrease in size the closer you get to h6; h1 being the largest size.

Syntax:
<h1> This is a sample heading. </h1>
<p> This is a sample paragraph. </p>

Images and Image Captions

Images in HTML can be added from your local device or from another website using the <img> tag. It is one of the tags that don't require a closing tag. You can place the image tags within or outside of a <figure> tag.

The most commonly used attributes with the <img> tag are:

  • src – Specifies the image source (URL or file path).
  • alt – Provides alternative text for the image if it can't be displayed.
  • width – Sets the width of the image in pixels or percentage.
  • height – Sets the height of the image in pixels or percentage.
  • title – Shows extra information about the image when you hover over it.
Syntax:

<img href= "https://www.samplesource.jpg" alt="Sample Picture" width="200" height ="200">

Tables

Tables are used to display data in a structured format. We use the <table> tag to create a table. The <tr> tag is used to create a table row. The <th> tag is used to create a table header. The <td> tag is used to create a table cell.

Lists

HTML supports several types of lists that help organize content. The two primary types are unordered lists (bulleted lists) and ordered lists (numbered lists). Lists are helpful for structuring information clearly.

Example:

Unordered List

  • Item One
  • Item Two
  • Item Three

Ordered List

  1. First Step
  2. Second Step
  3. Third Step

Use the <ul> tag for unordered lists and <ol> for ordered lists. Each list item goes inside a <li> tag.

    • href: Specifies the URL of the link
    • target: Specifies where to open the linked document

    Forms in HTML

    Forms are used in HTML to collect user input and send it to a server for processing. They enable interactive web pages.

    Common use Cases:

    • User Authentication- Login and registration.
    • Data Collection- Contact forms, surveys, feedback.
    • E-Commerce- Checkpuut, payment, shipping.
    • Search- Search boxes.
    • Settings/Preference- User profiles, account settings.
    • File Uploads- Document and image uploads.
    • Subscriptions- Newsletter signups, email lists.

    How forms work:

    • User Input- Fields like text input, checkboxes, dropdowns, etc.
    • Validation- Check data before submission (client-side).
    • Submission- Send data to a server (via GET or POST).
    • Processing- Server processes the data (save to database, send email, etc.)
    • Response- Show a success message or redirect

    IFrames

    What is an IFrame?

    An IFrame (Inline Frame) is n HTML element that embeds another HTML document inside the current page. It creates a window within your page that displays content from another source.

    We use the <iframe> tag to create an iframe.

    Common Use Cases:

    • Video Embedding- Embedding YouTube, Vimeo, or other video content
    • Maps- Displaying Google Maps or other mapping services
    • Social Media- Embedding Twitter feeds, Facebook posts, Instagram content
    • Documents- Displaying PDFs or other documents
    • Widgets- Including interactive widgets from other websites
    • Third-party Content- Embedding content from external sources

    Common IFrame Attributes:

    • src: Specifies the URL of the page to embed
    • width: Sets the width of the iframe (in pixels or percentage)
    • height: Sets the height of the iframe (in pixels or percentage)
    • title: Provides a descriptive title for accessibility
    • frameborder: Controls the border around the iframe (0 = no border, 1 = border)
    • allowfullscreen: Allows the iframe content to enter fullscreen mode
    • sandbox: Applies security restrictions to the iframe content
