Learning JavaScript

Introduction to JavaScript

JavaScript is a high-level, multi-paradigm programming language primarily known for enabling interactive and dynamic content on web pages.It can update both HTML and CSS. Additionally, it can calculate, manipulate, and validate data.

Main Purposes

  1. Makes websites more interactive
    • Responds to user actions (clicks, typing, scrolling)
    • Updates content without reloading the page.
    • Creates interactive elements like buttons, forms, and animations.
  2. Manipulates web pages
    • Changes HTML content dynamically
    • Modifies CSS styles
    • Shows/ hides elements
    • Updates text, images, and structure
  3. Handles user input
    • Validates forms before submission
    • Processes button clicks
    • Responds to keyboard and mouse events
  4. Communicates with servers
    • Sends and receives data (AJAX/fetch)
    • Updates pages with new information
    • Works with APIs
  5. Creates dynamic content
    • Builds interactive games
    • Creates animaions and transitions
    • Builds single-page applications (SPAs)

How to Run JavaScript Code

JavaScript can be added within the HTML file or it can be externally linked into your file using the <script> tags.

Code Syntax: 
<!-- This is adding script within your HTML Document -->
<!DOCTYPE>
<html lang ="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <script>
        console.log ("Hello World");
        // Insert JavScript Here
    </script>
    <!-- '//' writes comments in JavaScript -->
</body>
</html>


<!-- This is linking external script to your HTML Document -->
<!DOCTYPE>
<html lang ="en">
    <head>
        <meta charset="UTF-8">
        <title>Document</title>
    </head>
    <body>
        <script src ="script.js"></script>
        <!-- External Link to JavaScript file script.js -->
    </body>
    </html>

Save and open your file in a web browser. Then, open the browser's console:

  1. Right-click on the blank webpage.
  2. Click on "Inspect" or "Inspect Element" to open the Developer Tools.
  3. Find and select the "Console" tab, where you should see the output of the console.log statement.
console.log () is the command that prints something to the developer console in the browser.

Variables and Operators

Variables

Variables are the building blocks of any program. You can think of variables as "storage containers" for data in your code.

You can declare variables using the let keyword.

Code Syntax: 
let firstName = "John";
let lastName = "Doe";
                
console.log (firstName);
console.log (lastName);
You can also re-assign variables: 
let age = 11;
console.log(age); // output 11

age = 54;

console.log(age); // what will be the output now?
Notice the lack of let on line 4.
